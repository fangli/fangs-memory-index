# library/AGENTS.md — Generated Knowledge Output

This subtree is for generated knowledge records.

Agents may write here only when executing an approved runbook such as daily ingestion or manual import.

Recommended future memory search paths are selected subfolders here, not the project root.

## Subfolders

- `sources/` normalized source records.
- `atoms/` extracted reusable atom records.
- `views/` generated rollups.
- `manifests/` seen IDs, hashes, ingestion state.
- `indexes/` deterministic local indexes such as SQLite.
- `reports/` librarian reports.
